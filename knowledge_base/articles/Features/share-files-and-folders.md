# Share files and folders

## Share a link

1. Right-click a file or folder → **Share → Copy link**.
2. Anyone with the link can **view and download** by default.
3. (Plus and above) Set a **password** or an **expiry date** on the link.

## Invite collaborators

1. Right-click → **Share → Invite people**.
2. Enter their email and choose a permission level:

| Permission | Can view | Can edit | Can re-share |
|---|---|---|---|
| Viewer | yes | no | no |
| Editor | yes | yes | no |
| Manager | yes | yes | yes |

Collaborators need a Nimbus account to edit; viewers do not.

## Stop sharing

Open the item's **Share** dialog and click **Remove** next to a person, or
**Disable link** to revoke link access. Changes apply immediately.
