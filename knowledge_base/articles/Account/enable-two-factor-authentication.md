# Enable two-factor authentication

Two-factor authentication (2FA) adds a one-time code on top of the password.

## Set it up

1. Open **Settings → Security → Two-factor authentication**.
2. Click **Enable** and scan the QR code with an authenticator app
   (Google Authenticator, Authy, 1Password, etc.).
3. Enter the 6-digit code to confirm.
4. **Save the recovery codes** shown on screen — there are **10** of them, each
   usable once.

## Lost access to the authenticator

- Use one of the **recovery codes** at sign-in.
- No recovery codes? The account owner must verify identity by email before
  support can disable 2FA. This can take up to **2 business days**.

> Recovery codes are the only offline way back in. Tell customers to store them
> somewhere safe, not inside Nimbus itself.
