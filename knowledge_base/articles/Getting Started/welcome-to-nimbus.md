# Welcome to Nimbus

Nimbus is cloud file storage that syncs your files across all your devices. This
article is the starting point for new customers.

## What you get

- Files stored securely in the cloud and synced to every signed-in device.
- A desktop app (Windows, macOS, Linux), mobile apps (iOS, Android), and a web app.
- File sharing via links or by inviting collaborators.

## First steps

1. Create an account at **nimbus.example/signup** with your email and a password.
2. Verify your email — we send a confirmation link valid for **24 hours**.
3. [Install the desktop app](install-the-desktop-app.md) to start syncing.
4. (Recommended) [Enable two-factor authentication](../Account/enable-two-factor-authentication.md).
